# Hotel Reservation System

## Overview

Hotel Reservation System is a Java console-based application developed as part of the CodeAlpha Java Programming Internship. It allows users to view available rooms, book rooms, cancel reservations, and manage room availability.

## Features

* View available rooms
* Book a room
* Cancel room booking
* View room status
* Menu-driven interface

## Technologies Used

* Java
* ArrayList
* Classes and Objects
* Conditional Statements
* Loops
* Object-Oriented Programming (OOP)

## How It Works

1. User views available rooms.
2. User books a room.
3. Room status changes to booked.
4. User can cancel the booking.
5. Room becomes available again.
